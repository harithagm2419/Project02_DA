# Project02_DA
Showcasing the Customer Behavior analysis using Python, SQL, and Power Bi. 

## Data Analytics Project

### Overview

This project demonstrates an end-to-end data analytics workflow — from data loading and exploration to building an interactive Power BI dashboard and generating actionable insights. The goal is to extract meaningful patterns from data and present them in a clear, business-friendly format.

### Dataset

Source: Kaggle Dataset

Format: CSV

#### Description:
The dataset contains structured data used for analysis, including key variables relevant to business insights.

Tools & Technologies

Python (Pandas)

SQL (PostgreSQL)

Power BI (Dashboard creation & visualization)

### Jupyter Notebook

Steps Performed
1. Data Loading

Imported dataset using Python and SQL connections

Verified data structure and formats

2. Data Cleaning

Handled missing values

Removed duplicates

Corrected data types

Standardized column names

3. Exploratory Data Analysis (EDA)

Performed statistical summaries

Identified trends and patterns

Visualized distributions and relationships using charts

4. SQL Analysis

Queried data using joins, aggregations, and filters

Extracted insights directly from the database

Optimized queries for performance

5. Dashboard Creation (Power BI)

Built an interactive dashboard

Added filters, slicers, and KPIs

Visualized key business metrics

6. Reporting

Summarized findings in a structured report

Highlighted key insights and recommendations

Dashboard

Interactive Power BI dashboard showing:

Key performance indicators (KPIs)

Trends and comparisons

Filters for dynamic analysis
